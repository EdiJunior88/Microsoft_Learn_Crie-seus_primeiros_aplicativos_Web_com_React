Eventos e estado do React

Os aplicativos normalmente envolvem dados que serão alterados por interações do usuário. Essas alterações geralmente são gerenciadas por meio de alguma forma de estado e eventos. Neste módulo, exploramos os conceitos de estado e eventos. Também examinamos as ferramentas que o React fornece para o gerenciamento.

Objetivos de aprendizagem

Neste módulo, você vai:

  • Explorar os conceitos de estado e eventos.
  • Alterar os valores de estado.
  • Responder a eventos.
  • Usar ganchos para responder a atualizações.
