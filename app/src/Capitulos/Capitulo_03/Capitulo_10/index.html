Resumo

Neste módulo, exploramos como gerenciar o estado e os eventos no React. O React requer que o estado seja imutável. Portanto, precisamos substituir valores com estado por novos objetos. Uma forma de gerenciar esse processo é usando ganchos. Os ganchos nos fornecem acesso ao objeto de estado e a uma função que podemos usar para atualizá-lo.

Também vimos como adicionar eventos a um componente, permitindo que o usuário interaja com os dados na página. Terminamos o módulo usando o gancho de efeito para executar o código em resposta a uma atualização no estado.
