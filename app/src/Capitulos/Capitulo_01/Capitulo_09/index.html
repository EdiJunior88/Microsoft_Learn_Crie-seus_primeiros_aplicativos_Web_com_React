Resumo

O React é a estrutura JavaScript de front-end mais popular. Você pode usá-lo para criar aplicativos baseados em navegador e instalados localmente. O React introduz uma nova sintaxe chamada JSX para controlar a interface do usuário.

Começamos este módulo explorando os principais conceitos do React. Vimos como o React pode ajudar você a criar aplicativos Web. Exploramos as opções de instalação e como usar um empacotador para gerenciar nossos arquivos JSX. Depois de criar nosso primeiro projeto e os componentes, executamos o código localmente. Também mencionamos outros tipos de aplicativos que o React pode ajudar a criar.

Você aprendeu a:

  • Entender os principais conceitos do React.
  • Criar um aplicativo do React.
  • Criar um componente.
